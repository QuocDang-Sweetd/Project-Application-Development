
Cấp 0: 
Cài thư viện: pip install fastapi uvicorn
Chạy ứng dụng FastAPI : uvicorn main:app --reload (chạy trong terminal)
Kiểm tra server có chạy không:  http://127.0.0.1:8000/
Kiểm tra endpoint /health : http://127.0.0.1:8000/health
Kiểm tra Swagger UI : http://127.0.0.1:8000/docs
